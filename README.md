# Supermarket-Sales-Analysis-PowerBI
# 全球超市销售与利润多维度诊断看板 (Power BI)

## 1. 项目背景
本项目基于超市销售数据，利用 Power BI 构建了交互式决策看板。通过对销售额、利润占比、客户群体及地理维度的深度拆解，辅助管理者快速定位业绩增长点及潜在经营风险。

## 2. 核心看板展示
### A. 销售分析页
分析各客群（Consumer, Corporate, Home Office）的销售额分布，并监控年度销售额的时间序列增长趋势。
![销售分析](image_2d6f85.png)

### B. 产品分析页
利用树状图（Treemap）展示客群占比，并通过堆叠面积图监控 Top 5 热门产品的销售走势。
![产品分析](image_2d6c7c.png)

### C. 客户与区域分析页
聚焦核心城市（如 New York City）的销售贡献，下钻分析各区域（East, Central, West, South）的季度增长动能。
![客户分析](image_2d6c23.png)

## 3. 技术实现细节
* **数据清洗：** 使用 Power Query 处理日期格式并清理冗余字段。
* **DAX 指标开发：** 编写了销售额占比、同比增长率、利润率等核心业务指标。
* **交互体验：** 实现了日期切片器（Slicer）与跨图表联动，支持按年/季/月进行动态筛选。

## 4. 业务洞察建议
* **客群侧：** Consumer（消费者）群体贡献了超过 50% 的销售额，是公司最核心的收入来源。
* **区域侧：** East（东部地区）在第四季度展现出强劲的爆发力，建议针对该季节性趋势加大促销投入。
